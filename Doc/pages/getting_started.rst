Getting Started
===============

MDANSE stands for Molecular Dynamics Analysis for Neutron Scattering
Experiments. It is a tool for analyzing molecular dynamics
simulations, also in the context of neutron scattering experiments.

MDANSE Installation 
--------------------

To get started with MDANSE, you can follow the MDANSE installation guide
at :ref:`installation-tutorial`.


Explore the Documentation
--------------------------

MDANSE documentation is split into four sections:

- **Explanations** of MDANSE's capabilities and underlying theory.
- **Tutorials** with input files and reference results provided.
- **How-to Guides**: Practical solutions for common tasks.
- **Technical References**: information about MDANSE's features and functionalities.

Sample Data
-----------

MDANSE provides sample datasets that you can use for testing and learning.
You can find sample data and tutorials within the
`MDANSE-Examples GitHub repository <https://github.com/ISISNeutronMuon/MDANSE-Examples>`_.
